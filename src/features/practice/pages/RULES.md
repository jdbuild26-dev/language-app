# RULES

This directory (`src/features/practice/pages`) contains the **Main Practice** activities for the application.

**CRITICAL RULE: DO NOT CONFUSE WITH VOCABULARY PRACTICE.**

- Main practice activities are meant to be untouched unless explicitly requested by the user for "Main Practice".
- **DO NOT** apply vocabulary practice rules (like translating options on submit) to these files.
- Vocabulary practice activities are located in `src/features/vocabulary/pages/`.
